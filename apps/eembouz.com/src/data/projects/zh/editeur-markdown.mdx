## 编辑器设计

本项目旨在构建一款现代 Markdown
编辑器，将纯文本的简洁性与所见即所得界面的强大功能相结合。选择
__Tiptap__ 是因其灵活性及与 React 的无缝集成。界面设计
__直观且简约__，为技术写作者和开发者提供流畅愉悦的写作体验。

## 自定义工具栏与 Markdown 快捷键

编辑器包含__自定义工具栏__
，用户可轻松应用加粗、斜体、标题、列表和引用等格式。同时支持原生
__Markdown 快捷键__，实现快速自然的写作。所有 UI 组件使用
__SCSS__ 样式，确保视觉一致性和跨屏幕尺寸的响应式表现。
